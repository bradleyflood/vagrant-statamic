vagrant-statamic
================

A `Vagrant Up` for quick Statamic development.

### Provisions

- Ubuntu 'Precise32'
- Apache2
- PHP5 (latest)
- Configured php.ini and apache2.conf

### Requirements

- VirtualBox (latest)
- Vagrant (latest)

### Notes

Uses modified Bash script from [JeffreyWay/Vagrant-Setup](https://github.com/JeffreyWay/Vagrant-Setup)
